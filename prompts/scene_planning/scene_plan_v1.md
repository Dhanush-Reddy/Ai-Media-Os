# Scene Plan Prompt v1

Create a strict scene plan from an approved script.

Each scene must include:

- Sequential scene number
- Start time and duration in seconds
- Narration
- Visual type
- Visual description
- Image prompt when a generated image or diagram is needed
- Camera motion
- Transition
- Caption style
- Source claim IDs when the narration maps to research claims

Rules:

- Use only original generated visuals or rights-cleared visual directions.
- Do not request logos, copyrighted characters, or arbitrary web images.
- Scene numbers must start at 1 and not skip.
- Scenes must not overlap.

# Script Fact Check Prompt v1

Compare a script against project claims and sources.

Return:

- Verified claims mentioned
- Unverified claims mentioned
- High or critical research anchors missing from the script
- Whether the script is safe to proceed to scene planning

Rules:

- Treat unverified, disputed, contradicted, or unsupported claims as blockers when they appear as factual narration.
- Prefer conservative review when evidence is incomplete.

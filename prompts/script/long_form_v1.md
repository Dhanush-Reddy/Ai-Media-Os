# Long-Form Script Prompt v1

Write a YouTube script for the AI & Future channel using only verified research claims as factual anchors.

Required structure:

- Hook
- Main Story
- Context and Stakes
- Review Boundaries
- Outro

Rules:

- Do not present unverified, disputed, or contradicted claims as facts.
- Keep wording clear for a technical but non-specialist viewer.
- Preserve revision feedback as explicit guidance for a new version.
